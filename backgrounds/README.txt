Optional real-photo backgrounds
===============================

Every soundscape has a matching scene background built in as an
illustration, so the app works offline with no downloads.

To use a real photo instead, drop a JPEG in this folder with the matching
name — it automatically replaces the illustration:

  seaside.jpg          → Seaside
  forest.jpg           → Forest Walk
  library.jpg          → Library
  fire.jpg             → Fireplace
  rain.jpg             → Rainy Window
  white-noise.jpg      → White Mist
  pink-noise.jpg       → Pink Dusk
  brown-noise.jpg      → Sepia Dunes
  light-rain.jpg       → Light Rain
  ocean.jpg            → Open Ocean
  wind.jpg             → Windy Fields
  campfire-sea.jpg     → Campfire Shore
  forest-night.jpg     → Night Forest
  library-talking.jpg  → Reading Room
  rain-thunder.jpg     → Thunderstorm
  traffic.jpg          → City Lights

Use dark or moody photos for best readability; the app dims them slightly
so the timer stays legible. After adding photos, rebuild the Mac app
(mac/build.sh) so they get bundled into Focus.app as well.
